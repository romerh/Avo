Cypress Project with Typescript

Tests Included:
cart_tests - 3 tests of cart widget.
register_to_Avo- flow. test of registration process.
search_by_category- searching for a specific category.

Tools:
VScode IDE
Node
Cypress

Requirements
In order to utilise this project you need to have the following installed locally:
node version v14.17.0.
Chrome Browser version 94.
An IDE of youre choice(preferebaly VScode)that supports TS.
cypress node module installed.

Execution:
go to terminal (cmd or in iDE), make sure you are on the project location, enter "npx cypress open" in termianl and press enter,
select desired test for execution(in the Integration Tests section undet "Tests"), execution will start once selected.
